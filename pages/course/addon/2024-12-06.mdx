## 一、判断题

#### 1、第一台电子计算机ENIAC是在1946年诞生的。

答案：正确。

解析：ENIAC（Electronic Numerical Integrator And Computer）是世界上第一台通用电子计算机，确实在1946年在美国宾夕法尼亚大学首次公开亮相。

#### 2、摩尔定律指出，集成电路上可容纳的晶体管数目大约每18个月便会增加一倍。

答案：正确。

解析：摩尔定律是由英特尔创始人之一戈登·摩尔提出的，它描述了集成电路的性能随着技术进步而指数增长的趋势。

#### 3、互联网的前身ARPANET最初是为了商业用途而设计的。

答案：错误。

解析：ARPANET（阿帕网）是美国国防部高级研究计划署开发的一个实验性网络，最初目的是为了军事通信的可靠性和资源共享。

#### 4、TCP/IP协议是互联网的核心协议。

答案：正确。

解析：TCP/IP（传输控制协议/因特网互联协议）是支撑现代互联网的技术基础，负责数据包在网络上的传输。

#### 5、信息社会的发展导致了个人隐私泄露等问题。

答案：正确。

解析：随着信息技术的快速发展，个人信息保护成为了一个重要的社会问题，不当的信息收集和使用可能导致隐私泄露。

#### 6、计算机的第一代主要使用晶体管作为逻辑元件。

答案：错误。

解析：计算机的第一代（约1940年代中期到1950年代末期）主要使用真空管作为逻辑元件，而第二代则开始采用晶体管。

#### 7、计算机硬件包括中央处理器、内存、输入输出设备等组成部分。

答案：正确。

解析：计算机硬件是指组成计算机物理实体的各种设备，如CPU、RAM、硬盘、键盘、显示器等。

#### 8、Windows、Linux和macOS都是常见的计算机操作系统。

答案：正确。

解析：操作系统是管理计算机硬件与软件资源的程序，上述三种系统分别是微软、开源社区和苹果公司开发的操作系统。

#### 9、二进制数系统由0和1两个数字组成。

答案：正确。

解析：二进制是一种基于2的计数系统，只使用0和1两个符号来表示数值。

#### 10、ASCII码是一种用于表示英文字符的标准编码。

答案：正确。

解析：ASCII（American Standard Code for Information Interchange）是最早的一种字符编码标准，主要用于表示英文字符。

#### 输入格式

输入两个整数$$ x,n$$ (表示从周 $$x$$ 算起，经过 $$n$$ 天）。

#### 输出格式

输出一个整数，表示小鱼累计游泳了多少公里。

#### 输入输出样例

##### 输入1

```
3 10
```

##### 输出1

```
2000
```

> 输入表示从第 3 周起，经过 10 后，计算小鱼累计游泳了多少公里。计算方式为：
> 不是周末，循环 10 天，每次加 250，再输出结果。

#### 可AC代码

```python copy
x, n = map(int, (input().split()))
i = 1
KM = 0
while i <= n:  # 从第一天到第n天
    if x % 7 == 0 or x % 7 == 6:  # 判断是否是周六或者周日航程不加
        KM += 0
    else:  # 如果是周一到周五，则航程每天加250
        KM += 250
    i += 1
    x += 1

print(KM)
```

### 7. 讨厌的情人节

#### 题目描述
小明从出生到现在一直是单身，所以，他最恨情人节，不管是 `214` 还是 `77`，他都讨厌！
小明观察了 `214` 和 `77` 这两个数，发现：
$$2+1+4=7$$

如果一个整数符合下面三个条件之一，那么假定这个整数和 `7` 有关：
1.整数中某一位是 `7`；
2.整数的每一位加起来的和是 `7` 的整数倍；
3.这个整数是 `7` 的整数倍；

#### 输入输出格式

**输入格式**
输入包含两个正整数 `l,r`，表示给定的区间范围。
整数之间以空格间隔。
**输出格式**
请计算给定区间范围中和 `7` 无关的数字的平方和，并将结果对 $$10^9 + 7$$求模后输出。

#### 输入输出样例1
**输入**
`1 9`
**输出**
`236`

#### 输入输出样例2

**输入**
`10 11`
**输出**
`221`
#### 说明提示
$$1 \le l \le r \le 10^{18}$$

#### 可AC代码

```python copy
def is7(n):
    if n % 7 == 0:
        return True
    if sum([int(i) for i in list(str(n))]) % 7 == 0:
        return True
    if '7' in str(n):
        return True
    
    return False



l, r = map(int, input().strip().split())

res = 0
MOD = 10**9+7

for i in range(l, r+1):
    if not is7(i):
        res = (res + i**2) % MOD 

print(res)
```



### 8. 幸运单词

#### 题目描述
小明正在进行英语考试，但他的词汇量很小，所以每次做英语选择题的时候都很头疼。但是他找到了一种方法，经试验证明，用这种方法去选择选项的时候选对的几率非常大！
这种方法的具体描述如下：假设 `n` 是单词中出现次数最多的字母的出现次数，`m` 是单词中出现次数最少的字母的出现次数，如果 `$$n-m$$` 是一个质数，那么小明就认为这是个幸运单词，这样的单词很可能就是正确的答案。
聪明的你可以帮帮小明判断哪个单词是他的幸运单词吗？
#### 输入输出格式
**输入格式**
输入为一个单词字符串，其中只可能出现小写字母。
**输出格式**
输出共两行，第一行是一个字符串，假设输入的的单词是幸运单词，那么输出 `Lucky Word`，否则输出 `No Answer`；
第二行是一个整数，如果输入单词是幸运单词，输出 $$n-m$$ 的值，否则输出 `0`。

#### 输入输出样例1

**输入**
`error`
**输出**
`Lucky Word`
`2`
#### 输入输出样例2
**输入**
`olympic`
**输出**
`No Answer`
`0`

#### 说明提示

单词长度小于 `100`

#### 可AC代码

```python copy
def isPrime(n):
    if n == 1 or n == 0:
        return False
    if n == 2:
        return True
    
    for i in range(2, n):
        if n % i == 0:
            return False   
    return True

word = input()

cnt = [0 for i in range(26)]

for i in word:
    cnt[ord(i)-ord('a')] += 1

max_n = max(cnt)
min_n = 0
for i in cnt:
    if i > 0 and min_n == 0:
        min_n = i
    elif i > 0 and i < min_n:
        min_n = i

if isPrime(max_n-min_n):
    print('Lucky Word')
    print(max_n-min_n)
else:
    print('No Answer')
    print(0)
```



### 9. 给孩子分发饼干

###题目描述
假设你是一位很棒的老师，想要给你的孩子们一些饼干。但是，每个孩子最多只能给一块饼干。
对每个孩子` i`，都有一个胃口值 `child[i]`，这是能让孩子们满足胃口的饼干的最小尺寸；并且每块饼干 `j`，都有一个尺寸` size[j]` 。如果 `size[j] >= child[i]`，我们可以将这个饼干` j `分配给孩子 `i `，这个孩子会得到满足。你的目标是尽可能满足越多数量的孩子，并输出这个最大数值。

###输入格式
整数数组child与size
`1 <= g[i], s[j] <= 231 - 1`

###输出格式
一个整数

###输入样例
`[1,2,3]` \
`[1,1]`

###输出样例
`1`

###说明/提示
你有三个孩子和两块饼干，3个孩子的胃口值分别是：`1,2,3`。虽然你有两块饼干，由于他们的尺寸都是1，你只能让胃口值是1的孩子满足。所以你应该输出1。

#### 可AC代码

```python copy
child = eval(input())
size = eval(input())

idx_child, idx_size, cnt = 0, 0, 0

while idx_size < len(size):
    if size[idx_size] >= child[idx_child]:
        cnt += 1
        idx_size += 1
        idx_child += 1
    else:
        idx_size += 1

print(cnt)
```



### 10. 回文数字

#### 题目描述
一个向前或向后写的数字是一样的，称为回文数。例如，$$1234321$$ 为回文数字。所有的单位数都是回文数。非回文数可以通过一系列操作与回文数配对。首先，将非回文数字颠倒过来，并将结果添加到原始数字中。如果结果不是回文数，则重复此操作，直到给出回文数。例如，如果我们从 $$67$$ 开始，我们可以通过 $$2$$ 步得到回文数:$$67 + 76 = 143$$,$$143 + 341 = 484$$。给定任意正整数$$N$$，你要找出它的配对回文数以及找到它所需要的步骤数。

#### 输入格式
每个输入文件包含一个测试用例。每种情况由两个正数 $$N$$ 和 $$K$$ 组成，其中 $$N(\leq10^{10})$$ 为初始数，$$K(\leq 100)$$ 为最大步数。数字之间用一个空格隔开。

#### 输出格式
对于每个测试用例，输出两个数字，每行一个。第一个数字是成对的回文数 $$N$$，第二个数字是找到回文数的步骤数。如果 $$K$$ 步后没有找到回文数，只输出第 $$K$$ 步得到的回文数，而不是输出第 $$K$$步得到的回文数。

##### 输入1
`67 3`

##### 输出1
`484`
`2`

##### 输入2
`69 3`

##### 输出2
`1353`
`3`

##### 说明
无

#### 可AC代码

```python copy
def huiwen(n):
    return int(str(n)[::-1])

n, k = map(int, input().strip().split())
step = 0

reverse_n = huiwen(n)

while reverse_n != n:
    n = n + reverse_n
    reverse_n = huiwen(n)
    step += 1
    if step == k:
        break

print(n)
print(step)
```
